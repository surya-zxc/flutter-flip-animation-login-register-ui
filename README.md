# Flutter Login and Register UI with Flip Animation

Feel free to clone this UI and improve by your self

## Project Run

Clone this Project

  ```bash
  git clone 
  
  ```
  


Go to base
 ```bash

  cd my-project

 ```
 
Get Project Packages
```bash
  flutter pub get

```
Project Run
```bash
  flutter run
```
