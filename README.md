# Flutter Login and Register UI with Flip Animation

Feel free to clone this UI and improve it

## Project Run

Clone this Project

  ```bash
  git clone https://github.com/surya-zxc/flutter-flip-animation-login-register-ui.git
  
  ```
  


Go to base
 ```bash

  cd my-project

 ```
 
Get Project Packages
```bash
  flutter pub get

```
Project Run
```bash
  flutter run
```
